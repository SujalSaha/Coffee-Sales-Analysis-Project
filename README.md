# Coffee Sales Data Analysis - Spring Internship 2026

## Project Overview
This project performs an Exploratory Data Analysis (EDA) on coffee sales records to identify business trends and revenue drivers.

## Training & Methodology
During my internship at **IDEAS - ISI Kolkata**, I applied the following concepts:
- **Data Preprocessing:** Handling missing values and duplicates using Pandas.
- **Machine Learning Foundations:** Applying Linear Regression and Classification logic to sales data.
- **AI Architecture:** Understanding LLM training, tokenization, and model vs. system differences.

- Python (Google Colab)
